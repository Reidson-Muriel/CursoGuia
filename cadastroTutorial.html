Como criar Cadastro Simples
Apenas Html , Css e Js.

Vamos explicar o código HTML, CSS e JavaScript, passo a passo.O objetivo é criar um formulário de cadastro simples que permite
adicionar, editar e excluir entradas de uma lista.

Estrutura do Código

    • HTML: Define a estrutura da página.

    • CSS: Estiliza a aparência da página.

    • JavaScript: Implementa a lógica para interatividade (adicionar, editar e excluir).
