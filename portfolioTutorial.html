Como criar um projeto-Portfolio em HTML , CSS e Js.


HTML

Estrutura do Código

Detalhamento

    • head: Contém metadados, título e estilos CSS.

    • body: Onde o conteúdo visível do portfólio é definido.

    • div id="container": Um contêiner que centraliza o conteúdo.

    • div id="profile": Exibe a imagem do perfil e o nome do usuário.

    • div id="switch": Um botão para alternar entre modos claro e escuro.

    • ul: Lista de links para o portfólio e cursos.

    • div id="social-links": Links para redes sociais.

    • footer: Informações sobre direitos autorais.

CSS

      • Variáveis CSS: Define variáveis para cores, imagens de fundo e outros estilos.

      • Estilos gerais: Define estilos para todos os elementos, como fonte, cor de texto e margens.

      • Estilos específicos: Define estilos para cada seção do portfólio, como o perfil, switch, lista de links e redes sociais.

    JavaScript

      • toggleMode(): Função que alterna entre modos claro e escuro ao clicar no botão de switch.

      • classList.toggle('light') : Adiciona ou remove a classe light do elemento html..

      • querySelector("#profile img"): Seleciona a imagem do perfil.

      • setAttribute('src', 'img/avatar-light.png'): Altera a imagem do perfil para a versão clara ou escura.
