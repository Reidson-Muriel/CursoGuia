Como criar um Gerador-Senha em HTML , CSS e Js.

HTML

Estrutura do Código

Detalhamento

    • !DOCTYPE html: Declara que este é um documento HTML5.

    • html lang="pt-BR": Define o idioma da página como português do Brasil.

    • head: Contém metadados, como o título da página e links para arquivos CSS (neste caso, o CSS está embutido).

    • body: Contém o conteúdo visível da página.

    • div class="gerador": Um contêiner para centralizar e organizar os elementos do gerador de senhas.

    • h1>Gerador de Senha/h1: Título da página.

    • label for="comprimento"Quantidade da Senha:/label: Rótulo para o campo de entrada que especifica o comprimento da senha.

    • input type="number" id="comprimento" min="1" placeholder="Digite de 1 a 20" style="text-align: center;" /: Campo de entrada onde o usuário pode digitar o comprimento da senha desejada, com um valor mínimo de 1.

    • input type="checkbox" id="incluirNumeros": Caixa de seleção para incluir números na senha.

    • input type="checkbox" id="incluirSimbolos": Caixa de seleção para incluir símbolos na senha.

    • button onclick="gerarSenha()">Gerar/button: Botão que chama a função gerarSenha() quando clicado.

    • button onclick="copiarSenha()">Copiar/button: Botão que chama a função copiarSenha() quando clicado.

    • div id="resultado": Um contêiner para exibir a senha gerada.


CSS

Estrutura do Código

Detalhamento

    •body: Define a fonte, cor de fundo e centraliza o conteúdo na tela.

    •.gerador: Estiliza o contêiner do gerador de senhas com fundo branco, padding, bordas arredondadas e sombra.

    •h1: Define o estilo do título.

    •input[type="number"], input[type="checkbox"]: Adiciona margens para os campos de entrada e caixas de seleção.

    •button: Estiliza os botões, incluindo cores e efeitos de hover.

    •#resultado: Estiliza a área onde a senha gerada é exibida.


JavaScript

Estrutura do Código

Detalhamento

    •function gerarSenha(): Função que gera a senha com base nas opções selecionadas pelo usuário.

    •const comprimento = parseInt(document.getElementById('comprimento').value);: Obtém o comprimento da senha a partir do campo de entrada.

    •if (isNaN(comprimento) || comprimento > 20): Verifica se o comprimento é um número válido e se não excede 20.

    •const incluirNumeros = document.getElementById('incluirNumeros').checked;: Verifica se a opção de incluir números está selecionada.

    •const incluirSimbolos = document.getElementById('incluirSimbolos').checked;: Verifica se a opção de incluir símbolos está selecionada.

    •let caracteres = "abcdefghijklnmpqrstuvwxzABCDEFGHIJKLNMPQRSTUVWXZ";: Define os caracteres padrão da senha.

    •if (incluirNumeros) { caracteres += numeros; }: Adiciona números aos caracteres se a opção estiver selecionada.

    •if (incluirSimbolos) { caracteres += simbolos; }: Adiciona símbolos aos caracteres se a opção estiver selecionada.

    •for (let i = 0; i < comprimento; i++) {: Loop que gera a senha com o comprimento especificado.

    •document.getElementById('resultado').textContent=senha;: Exibe a senha gerada na área de resultado.

    •function copiarSenha(): Função que copia a senha gerada para a área de transferência.

    •navigator.clipboard.writeText(resultado): Usa a API de clipboard para copiar a senha.
